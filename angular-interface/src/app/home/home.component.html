Welcome to FRED

When publishing scientific studies, it is important to make all data and metadata available to promote scientific discovery and innovation. FRED (FaiR Experimental Designs) is a tool to help you generate metadata in a hierarchical structured YAML file using interactive dialogs.

Metadata

FRED metadata is divided into three sections:
  • Project — contains general information about the project like description or owner.
  • Experimental Setting — depicts the experimental design information including investigated conditions and associated samples.
  • Technical Details — stores technical information about the project like sequencing technique or analysis.
